# Reference

本區是平台的規格表。

凡是寫在 `Reference` 的定義，都是 notebooks、Product App、backend、Julia Runner 與核心能力需要對齊的 SoT。

<Aside type="note" title="How to read Reference">

先看 `Architecture`、`Notebooks`、`Core`、`Data Formats` 這些 owner layer，再用 `Product App` 查產品化 App contracts，用 `Guardrails` 確認開發流程與寫作規範。

</Aside>

## Reading Map
