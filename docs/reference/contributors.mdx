#Contributors

This document is the official record of contributors to the project. All names mentioned in the file Frontmatter (e.g. `owner`, `last_updated_by`) **must** exist in this directory.

<Aside type="note" title="How to use this page">

This page is not a thank you list, but an official registry for frontmatter name verification. This should be updated before adding or citing new names.

</Aside>

## Registry Map

| category | purpose |
| --- | --- |
| Primary Contributors | Core development, maintenance, document writing |
| Secondary Contributors | Provide advice or guidance, but do not directly write code/docs |
| Honorary Contributors | Formerly a major contributor, now retired from core maintenance |

## Agent Rule
```markdown
## Contributors Registry Rules
- **Name Validation**: Any person mentioned in Frontmatter (`owner`, `last_updated_by`) MUST exist in this Contributors registry.
- **Categories**:
  - Primary: Core developers and maintainers.
  - Secondary: Advisors who don't directly write code/docs.
  - Honorary: Former primary contributors.
- **Update Protocol**: Add new contributors before referencing them in any document.
```
