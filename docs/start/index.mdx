Start Here is the shortest research-first prototype path through the repo. It only introduces Julia Core + Pluto: first run a Pluto notebook, then understand the reusable circuit design model.

<Steps>
1. Install a minimal research environment so that Pluto can load local Julia packages.
2. Run through the first Pluto notebook and confirm that Julia Core, Visualizer and real solver path are available.
3. Understand reusable circuit design: Component Library -> CircuitPlan -> reusable plan builder -> schematic/export intent -> simulation.
</Steps>
