Product App files describe the product workbench: Frontend, Backend, Desktop shell, task lifecycle, ResultView and shared app semantics. It is not the main thread of Start Here; research and prototypes start with Pluto Notebook.

<Aside type="note" title="Notebook-first boundary">
Pluto Notebook is a straightforward study of cockpit. Python Notebook is the outpost verification surface of Product App, used to check the interaction between data, Backend API, task/result contract and future App needs; Python Analysis Core is a reusable analysis library, which may be called by Pluto through Analysis Bridge.
</Aside>
