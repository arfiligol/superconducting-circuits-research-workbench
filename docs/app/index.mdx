Product App 文件描述產品化 workbench：Frontend、Backend、Desktop shell、task lifecycle、ResultView 與 shared app semantics。它不是 Start Here 的主線；研究與 prototype 先從 Pluto Notebook 開始。

<Aside type="note" title="Notebook-first boundary">
  Pluto Notebook 是直接研究 cockpit。Python Notebook 是 Product App 的前哨驗證面，用來檢查資料、Backend API、task/result contract 與未來 App 需要的互動；Python Analysis Core 則是可重用 analysis library，可能被 Pluto 透過 Analysis Bridge 呼叫。
</Aside>
