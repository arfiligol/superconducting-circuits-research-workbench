# Managing Datasets

Use the Application Interface for routine dataset management.
Use Python Backend APIs or maintenance scripts for scripted changes.

## Usage

1. Open the Electron App.
2. Go to `Dataset`.
3. Search or filter the dataset list.
4. Open a dataset to inspect design scopes, traces, tags, and provenance.
5. Use `Raw Data` for trace previews.

## Deletion

Dataset deletion affects metadata and TraceStore references.
Keep deletion behind explicit backend validation and tests.

<Aside type="danger" title="Destructive operation">

Do not add ad hoc deletion scripts that bypass backend services.
The backend must preserve provenance guarantees and avoid orphaning TraceStore data.

</Aside>
