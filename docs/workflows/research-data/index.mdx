# Research Data & Evidence

Use this workflow when you need notebook-readable inputs and reproducible evidence for a study: HFSS exports, VNA exports, local arrays, fitted parameters, generated figures, and report tables.

## Minimum Evidence Bundle

| Item | Record |
| --- | --- |
| source files | path, generator, export date, and design label |
| axes | name, unit, ordering, and sweep range |
| transforms | normalization, coordinate transform, filtering, and fitting window |
| outputs | figure code, table code, and derived parameters |

<Aside type="tip" title="Keep provenance close">

Put the source path and unit assumptions next to the notebook cell that reads the data. A later report should not need to infer how a plotted trace was produced.

</Aside>
