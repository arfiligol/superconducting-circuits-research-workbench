# Archived Code Quarantine

This directory is inert reference material only.

Archived code is not part of the active product, Core, App, docs build, tests,
linting, type checking, package discovery, Sphinx autodoc, or runtime imports.
Active code must not import from this directory, and active documentation must
not link to it as a current implementation surface.

Agents may inspect archived material only when the user explicitly asks to
search, compare, or reference archived code.
